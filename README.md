Сайт-визитка для современной танцевальной студии. Проект реализован в рамках учебной дисциплины и личного портфолио для отработки навыков веб-разработки

О проекте
Цель: Создать удобный инструмент для привлечения новых учеников, презентации направлений танцев и записи на пробные занятия
Функционал:
Адаптивная верстка: Сайт корректно отображается на смартфонах, планшетах и ПК
Интерактивное расписание: Реализована сетка занятий с фильтрацией по направлениям
Форма записи: Сбор данных пользователей (имя, телефон, выбранное направление) с валидацией полей
Галерея: Слайдер с фотографиями преподавателей и танцевальных направлений

Стек технологий
Frontend: HTML5, CSS3 (Flexbox, Grid), JavaScript (ES6+)

Инструменты: Figma (создание макета), Git (контроль версий), VS Code

Аналитика и QA (что было сделано)
- Проведен анализ предметной области (услуги танцевальных студий)
- Спроектирована логика взаимодействия пользователя с формой записи
- Проведено тестирование верстки на различных разрешениях экрана и в разных браузерах (Chrome, Safari, Firefox)
